# HES
Repository for the HABET Electrical System (HES).  This repository will be for revision 3.0 of the new HES system that will be based on some of the Cubesat standards.

HES 3.0 will have the following features

* Utilize PC014 connector to allow compatability with other cubesat hardware
* Based on Raspberry Pi CM3 module
* Two on board cameras tied to Raspberry Pi CM3
* Access to I2C and serial data through PC014 connector to Raspberry Pi
* Access to unregulated battery voltage (nominal 7.4 VDC) through PC104
* Access to regulated 5V and 3.3V power busses through PC104
* Access to other GPIO lines to the Raspberry Pi
